# SmartInsuredReserve - Calculator (Next.js + TypeScript)

Minimal Next.js + TypeScript scaffold for a calculator demo.

Getting started

1. Install dependencies

   npm install

2. Start dev server

   npm run dev

3. Open http://localhost:3000


Project layout

- `pages/` - Next.js pages
- `src/components/Calculator.tsx` - Calculator UI
- `src/lib/calculator.ts` - Core calculator logic (pure functions)
- `styles/` - global CSS
- `tests/` - unit tests
