# SmartInsuredReserve

## Overview
Reserve calculation tool for insurance planning. Computes required reserves based on policy parameters and risk factors.

## Classification
- **Tier:** 2 — Interactive
- **Company:** Freebridge

## Tech Stack
- **Framework:** Next.js 14, React 18
- **Language:** TypeScript 5.4
- **Styling:** Tailwind CSS 3
- **Testing:** Jest, Testing Library
- **Package Manager:** npm

## Key Commands
- `npm run dev` — Start development server
- `npm run build` — Production build
- `npm test` — Run Jest tests
