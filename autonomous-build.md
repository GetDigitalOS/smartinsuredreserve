# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page calculator that optimizes insurance deductibles by building a self-insurance reserve fund. The prior phase is complete: the projection engine, insurance-domain helpers, input validation, localStorage scenario persistence, and the paginated projection table all exist with unit tests. This phase adds report export (CSV/print, the last open roadmap item), shared formatting/summary libraries, and the compliance report's "Important" fixes — reduced-motion support, error-boundary logging, a privacy page, and SEO structured data. Gate command for every chunk: `npm test`.
