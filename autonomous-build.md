# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund; prior phases shipped the projection engine, input validation, scenario save/load, CSV/print report export, reduced-motion handling, SEO/social/canonical metadata, JSON-LD, a hydration loading skeleton, Dependabot, a tightened CSP, and an enforced Jest test script — all merged and passing. Phase 5 pays down the roadmap tech debt and the remaining compliance quality gaps: centralizing the default input constants and deductible-tier configuration into dedicated modules, consolidating the projection engine onto the shared tier helper, enabling production browser source maps for observability, and adding a shared UI-copy module plus a first-use guidance component. Test command: `npm test`.
