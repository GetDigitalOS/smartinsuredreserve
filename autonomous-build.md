# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund; Phase 1 (merged to main) shipped the projection/validation/scenario engine, the security & observability hardening, the header-parity work, and the derived-data analytics/export layer (`src/lib/metrics.ts`, `src/lib/reserveHealth.ts`, `src/lib/scenarioCompare.ts`, `src/lib/reserveMilestones.ts`, `src/lib/growth.ts`, `src/lib/yearlyDelta.ts`, `src/lib/scenarioSummary.ts`, `src/lib/reportSummary.ts`, `src/lib/reportText.ts`, `src/lib/reportMarkdown.ts`, plus the detailed and metrics CSV builders and the `deriveMetricInsights`/`deriveHealthInsight`/`deriveMilestoneInsights`/`buildReportSummary` composers). Phase 2 starts from that fully green additive layer and continues it: new export/report renderers (HTML, delta CSV, metrics report, full report), two new composers over the existing growth analytics, a scenario-outcome diff, a growth insight, and a batch scenario import — all additive pure functions with deterministic Jest coverage so every existing test keeps passing. Test command: `npm test`.
