# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer projecting a self-insurance reserve fund; the prior phase completed a layer of pure derived-data analytics and export composers (`computeCompositionShare`, `classifyDeltaTrend`, `describeReserveProgress`, `buildGrowthTableCsv`, `buildGrowthTableMarkdown`, `deriveCompositionInsight`, `buildCompositionCsv`, `buildReportBundleJson`, `describeYearlyDeltas`, and `computeReserveTargetGap`). This phase extends that same layer with additional pure, deterministic label and export composers built strictly on top of the already-committed modules (`src/lib/compositionShare.ts`, `src/lib/deltaTrend.ts`, `src/lib/reserveTargetGap.ts`, `src/lib/growth.ts`, `src/lib/reserveMilestones.ts`, `src/lib/scenarioSummary.ts`, `src/lib/metrics.ts`, `src/lib/growthTable.ts`, `src/lib/summary.ts`, `src/lib/format.ts`, `src/lib/types.ts`) — every task adds a new file and a new test, so all existing tests keep passing. Test command: `npm test`.
