# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer projecting a self-insurance reserve fund; prior phases shipped the projection/validation/scenario engine, the security & observability hardening, the header-parity work, and the derived-data analytics layer (`src/lib/metrics.ts`, `src/lib/reserveHealth.ts`, `src/lib/scenarioCompare.ts`, `src/lib/reserveMilestones.ts`, the detailed CSV export, and the `deriveMetricInsights`/`deriveHealthInsight`/`buildReportSummary` composers). This phase deepens the derived-data and export layer on top of those modules: scenario JSON import, two new pure trend analytics (CAGR, year-over-year deltas), a metrics CSV, milestone insights, a scenario summary, and plain-text plus Markdown report builders — all additive pure functions with deterministic Jest coverage so existing tests keep passing. Test command: `npm test`.
