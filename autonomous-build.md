# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer projecting a self-insurance reserve fund; the prior phase completed the pure derived-data and report/export layer (scenario JSON import, `computeReserveCagr`, `computeYearlyReserveDeltas`, the plain-text and Markdown report builders, the scenario summary, the metrics CSV, and milestone insights). This phase deepens that same layer with additional pure, deterministic analytics and export composers built on top of the committed modules (`src/lib/growth.ts`, `src/lib/yearlyDelta.ts`, `src/lib/reportSummary.ts`, `src/lib/scenarioCompare.ts`, `src/lib/format.ts`): a CAGR insight, yearly-delta trend statistics, JSON and HTML report exports, a yearly-delta CSV, a comparison description, reserve-progress and reserve-composition analytics, and a combined growth table — all additive so existing tests keep passing. Test command: `npm test`.
