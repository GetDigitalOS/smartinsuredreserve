# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund; Phases 1 and 2 shipped and merged a layer of pure, deterministic derived-data analytics and report/export composers (e.g. `summarizeYearlyDeltas`, `computeReserveTargetGap`, `summarizeGrowthTable`, `describeDeltaTrend`, `buildCompositionShareCsv`, `buildGrowthTotalsCsv`). Phase 3 starts from that merged state and extends the same pure-function analytics/export layer with additional deterministic CSV composers, Markdown composers, summary labels, and insight lines built strictly on top of the already-committed base modules (`src/lib/deltaStats.ts`, `src/lib/reserveTargetGap.ts`, `src/lib/growthTableTotals.ts`, `src/lib/format.ts`, `src/lib/types.ts`, `src/lib/insights.ts`); every task is additive so all existing tests keep passing. Test command: `npm test`.
