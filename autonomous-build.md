# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund; Phase 1 shipped the projection engine, validation, scenarios, CSV/print export, SEO/social/canonical metadata, the DRY paydown onto `defaults.ts`/`deductibleTiers.ts`/`messages.ts`/`format.ts`, and the `InsightsPanel`/`SelectField`/`FirstUseNotice` components, all merged to main. Phase 2 closes the ranked security/compliance gaps from `docs/planning/NEXT_SESSION.md`: dropping the CSP `'unsafe-eval'` allowance by removing the dead `Function()` helper, adding HSTS, delivering headers at the Cloudflare edge via a `_headers` file, and adding client-side error capture (observability) wired into the app entrypoint and error boundary. Test command: `npm test`.
