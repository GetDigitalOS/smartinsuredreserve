# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund; Phases 1–3 shipped a layer of pure, deterministic derived-data analytics with CSV / Markdown / label composers on top of the base modules in `src/lib/` (e.g. `summarizeYearlyDeltas`, `computeReserveTargetGap`, `summarizeGrowthTable`, `computeReserveComposition`, `computeReserveProgress`, `findReserveMilestones`, `compareProjections`). Phase 4 starts from that merged state and rounds out the export layer for the base analytics that still lack full CSV / Markdown / label wrappers — `reserveComposition`, `scenarioCompare`, `reserveProgress`, and `reserveMilestones` — plus one composed summary line, every task additive and built strictly on already-committed modules so all existing tests keep passing. Test command: `npm test`.
