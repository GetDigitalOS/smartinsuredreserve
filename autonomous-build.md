# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer projecting a self-insurance reserve fund; the prior phase completed a layer of pure derived-data analytics and report/export composers (`deriveGrowthInsight`, `summarizeYearlyDeltas`, `buildReportJson`, `buildHtmlReport`, `buildYearlyDeltaCsv`, `describeComparison`, `computeReserveProgress`, `computeReserveComposition`, and `buildGrowthTable`). This phase extends that same layer with additional pure, deterministic analytics and export composers built strictly on top of the already-committed modules (`src/lib/reserveComposition.ts`, `src/lib/deltaStats.ts`, `src/lib/reserveProgress.ts`, `src/lib/growthTable.ts`, `src/lib/reportSummary.ts`, `src/lib/yearlyDelta.ts`, `src/lib/format.ts`, `src/lib/types.ts`): a composition-share ratio, a delta-trend classifier, human-readable progress and delta labels, growth-table CSV and Markdown exports, a composition CSV column set, a composition insight line, a target-gap helper, and a bundled JSON report — all additive so existing tests keep passing. Test command: `npm test`.
