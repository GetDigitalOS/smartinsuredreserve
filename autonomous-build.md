# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund. Phase 1 is merged: the projection engine, insurance helpers, input validation, localStorage persistence, paginated projection table, CSV/print export, shared formatting/summary libraries, error-boundary logging, reduced-motion support, a privacy page, and SEO structured data are all complete with passing unit tests. Phase 2 resolves the four outstanding compliance gaps in priority order: npm audit vulnerabilities (2 high, 2 moderate), the CSP `unsafe-eval` directive enabled by the now-unused `evaluateExpression` helper, security header delivery to the Cloudflare Pages edge (including HSTS), and missing client-side observability wired to the error boundary. Test command: `npm test`.
