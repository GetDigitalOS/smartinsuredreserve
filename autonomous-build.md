# smartinsuredreserve — Autonomous Build Backlog
SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund; Phase 1 shipped the projection engine, validation, scenarios, CSV/print export, and SEO/social/canonical metadata, and Phase 2 closed the security/compliance gaps (dropped CSP `'unsafe-eval'`, added HSTS, delivered edge headers via `_headers`, and wired client-side error capture into the app entrypoint and error boundary) — both phases are complete on `auto/build/smartinsuredreserve`. Phase 3 closes the remaining ranked Tier 2 compliance gaps from `docs/architecture/COMPLIANCE_REPORT.md` that need no L0 files: first-party privacy-respecting analytics, an explicit `Cache-Control` caching strategy, wiring the existing debounce utility into the calculator, and completing structured data with an `Organization` JSON-LD block. Test command: `npm test`.
