# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund on top of a layer of pure, deterministic derived-data analytics in `src/lib/`. Phases 1–4 shipped the base analytics modules (`classifyDeltaTrend`, `summarizeGrowthTable`, `findReserveMilestones`, `classifyReserveHealth`, `compareProjections`, `summarizeYearlyDeltas`) and, for several of them, CSV / Markdown / label composers. Phase 5 starts from that merged state and completes the export trio for the base analytics that still lack full wrappers — `deltaTrend`, `growthTableTotals`, `reserveMilestones`, and `reserveHealth` — plus two composed summary lines, every task additive and built strictly on already-committed modules so all existing tests keep passing. Test command: `npm test`.
