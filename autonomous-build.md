# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund; Phase 1 shipped and merged a layer of pure, deterministic derived-data analytics and report/export composers (e.g. `computeReserveComposition`, `computeCompositionShare`, `classifyDeltaTrend`, `describeReserveProgress`, `buildGrowthTable`, `buildGrowthTableCsv`, `buildGrowthTableMarkdown`, `buildCompositionCsv`, `deriveCompositionInsight`, and `buildReportBundleJson`). Phase 2 starts from that merged state and extends the same pure-function analytics/export layer with additional deterministic labels, CSV/Markdown composers, aggregate summaries, and insight lines built strictly on top of the already-committed modules (`src/lib/compositionShare.ts`, `src/lib/deltaTrend.ts`, `src/lib/progressLabel.ts`, `src/lib/growthTable.ts`, `src/lib/reserveComposition.ts`, `src/lib/format.ts`, `src/lib/types.ts`, `src/lib/insights.ts`, `src/lib/csv.ts`); every task is additive so all existing tests keep passing. Test command: `npm test`.
