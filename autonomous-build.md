# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund; Phase 1 (merged to main) shipped the pure derived-data and report/export layer — `computeReserveCagr`, `computeYearlyReserveDeltas`, `summarizeYearlyDeltas`, `buildReportSummary`, the JSON/HTML/text/Markdown report builders, the metrics and yearly-delta CSVs, `compareProjections`/`describeComparison`, `computeReserveProgress`, and `computeReserveComposition`. Phase 2 continues that same additive, deterministic analytics-and-export layer on top of the committed modules (`src/lib/deltaStats.ts`, `src/lib/reserveProgress.ts`, `src/lib/reserveComposition.ts`, `src/lib/growth.ts`, `src/lib/csv.ts`, `src/lib/reportHtml.ts`, `src/lib/reportJson.ts`, `src/lib/format.ts`, `src/lib/types.ts`): human-readable describe strings, composition shares, a growth multiple, a composition CSV, composition HTML/JSON exports, and a reserve-shortfall calculator — every change additive so existing tests keep passing. Test command: `npm test`.
