# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund. Phases 1 and 2 are merged: the projection engine, validation, persistence, table, CSV/print export, privacy page, and SEO JSON-LD shipped in Phase 1, and Phase 2 closed the four critical security gaps (npm audit clean, `unsafe-eval` removed from the CSP, `public/_headers` edge delivery with HSTS, and client-side observability wired into the error boundary). Phase 3 works through the compliance report's remaining "Important" and "Recommended" gaps: design-system color tokens, a reduced-motion CSS fallback, icon accessibility, a reusable form field, an input debounce utility, and SEO completeness (robots.txt, sitemap.xml, Open Graph / Twitter social tags). Test command: `npm test`.
