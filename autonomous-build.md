# smartinsuredreserve — Autonomous Build Backlog

SmartInsuredReserve is a Next.js 14 / React 18 / TypeScript single-page insurance deductible optimizer that projects a self-insurance reserve fund; Phase 1 shipped the projection/validation/scenario engine plus the security & observability hardening (dead `Function()` removed, CSP `unsafe-eval` dropped, HSTS added, an edge `_headers` file, and `src/lib/observability.ts` wired into `pages/_app.tsx` and `src/components/ErrorBoundary.tsx`), all merged to main. Phase 2 deepens the observability thread (richer captured-error records, a read accessor, and capture on the client-side export path in `ReportActions`) and closes the next-runtime-vs-edge security-header parity gap between `next.config.js` and `_headers`.
